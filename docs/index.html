Meet your new Companion.

A tiny always-on-top overlay that brings animated companions to the Claude Desktop app. They wander, they wave, they nap. Pick them up. Throw them around.

Built to stay out of your way

Claude Desktop is never touched. Companions watch the window and react.

Per-pixel click-through

Clicks on empty space pass straight to Claude. Only the companion itself is interactive.

Claude-aware

Appears when Claude is frontmost. Hides when you switch to Chrome. Quits when Claude quits.

Drag, throw, watch it land

Click and hold to pick a companion up. Let go and gravity pulls them back to Claude's bottom edge.

Five packs shipped

Pane, Blob, Cat, Ghost, Sprite. Each a self-contained folder: SVG + animations + speech.

Personality presets

Calm, Normal, Playful. Sliders for activity frequency, walk speed, chattiness, and quiet hours.

macOS native

No Dock icon, no app switcher. Runs on Tauri with NSWorkspace and CGWindow — no accessibility permission required.

Install & run

Requires Rust + Node. macOS only for now.

git clone https://github.com/ben4mn/companion.git
cd companion
pnpm install
pnpm tauri dev

First build compiles the Rust tree (~3–5 min). Pane will snap to the bottom-right of your Claude Desktop window.
